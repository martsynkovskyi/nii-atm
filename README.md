# НИИ Атмосфера - проект страницы услуг

Статический проект для демонстрации новой группировки страницы услуг.

## Публикация на GitHub Pages

1. Создать репозиторий.
2. Загрузить содержимое этого архива в корень репозитория.
3. В настройках GitHub Pages выбрать публикацию из ветки `main`, папка `/root`.
4. Для поддомена использовать CNAME: `nii-atm.martsynkovskyi.ru`.
