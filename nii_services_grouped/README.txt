Проект группировки страницы services.html

Содержимое:
- services_grouped_preview.html — самостоятельный HTML-прототип страницы.
- services_grouped_fragment.html — фрагмент для вставки в текущий шаблон WordPress вместо существующего блока grid-blocks.
- assets/services — новые изображения карточек услуг.

Идея внедрения:
1. Сохранить существующую двухколоночную сетку .grid-blocks.
2. Между группами добавить заголовки направлений .service-group__title и краткие описания.
3. Заменить изображения карточек на новые.
4. Сохранить существующие ссылки на страницы услуг.
